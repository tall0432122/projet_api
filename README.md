# Projet de Nettoyage CSV

## Description
API Flask pour nettoyer automatiquement les fichiers CSV : suppression des doublons, traitement des valeurs manquantes et outliers.

## Installation
```bash
python -m venv venv
source venv/bin/activate  # ou venv\Scripts\activate sous Windows
pip install -r requirements.txt
python app.py
```

## Utilisation
1. Ouvrir http://localhost:5000
2. Uploader un fichier CSV
3. Télécharger le résultat nettoyé
